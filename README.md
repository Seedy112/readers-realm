# Reader's Realm

Reader's Realm is a full-stack web application for book lovers to discover books,
write reviews, participate in discussions, and join reading events.

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MySQL  
Authentication: JWT  

## Features
- User registration and login
- Book catalog with reviews and ratings
- Discussion forum and comments
- Event management
- User profiles
